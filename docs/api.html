Tickets

Ticket operations

List tickets

Returns tickets. Supports filtering, pagination, sorting, field filtering, counting and enveloping.

Authorizations:
BearerAuthBasicAuth
query Parameters
fields
Array of strings

Restrict response to the specified fields (comma-separated).

count
boolean
Default: false

If true, returns Total-Count header.

envelope
boolean
Default: false

If true, wraps the result in an envelope and always returns HTTP 200.

page
integer >= 1
Default: 1

Page number starting from 1.

per_page
integer [ 0 .. 100 ]
Default: 10

Items per page (0–100, default 10).

sort
string
Default: "-created_at"
Enum: "updated_at" "-updated_at" "created_at" "-created_at" "user_id,-updated_at"

Sort order.

id
Array of strings

Comma-separated list of ticket ids.

inbox_id
Array of strings

Comma-separated list of inbox ids.

state
Array of strings
Items Enum: "open" "hold" "closed"

Comma-separated list of ticket states to filter.

user_id
Array of strings or null

Comma-separated list of user ids. Use null to find unassigned tickets.

label_id
Array of strings

Comma-separated list of label ids.

type
Array of strings (TicketType)
Items Enum: "email" "twitter" "twitter_dm" "phone" "chat" "web" "sms" "whatsapp" "fb_messenger" "instagram_dm" "call"

Comma-separated list of ticket types.

spam
boolean

Filter by spam flag.

trash
boolean

Filter by trash flag.

since_created_at
string <date-time>

Filter to resources created at or after this UTC ISO-8601 timestamp.

since_updated_at
string <date-time>

Filter to resources updated at or after this UTC ISO-8601 timestamp.

Responses

Primary server (replace {site} with your help desk identifier)

https://{site}.enchant.com/api/v1/tickets
